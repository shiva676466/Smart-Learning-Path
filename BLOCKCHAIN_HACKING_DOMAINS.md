
Users can now generate personalized learning roadmaps for:

- Ethical Hacking & Security (Beginner → Advanced)

Example roadmap generation options:

- **Blockchain Beginner (30 days, 2 hrs/day)** → 60 hours of structured learning
- **Hacking Intermediate (60 days, 3 hrs/day)** → 180 hours of hands-on practice

### 3. Track Progress

- Each domain has 10 topics per level
- Topics scale with user's available hours
- XP rewards based on difficulty and workload
- Progressive difficulty from beginner to advanced

## 🚀 Getting Started

### Prerequisites

- Python 3.8+
- Django 4.2+
- Virtual environment activated

### Installation Steps

```bash
# 1. Pull the branch
git checkout feature/add-blockchain-hacking-domains

# 2. Apply migrations
python manage.py migrate roadmap

# 3. Seed the skills
python manage.py seed_skills

# 4. Create a superuser (if not exists)
python manage.py createsuperuser

# 5. Run the dev server
python manage.py runserver

# 6. Visit admin panel and verify
# http://localhost:8000/admin/
```

### Testing

1. **As a superuser in Django admin:**
   - Navigate to Roadmap → Skills
   - Verify 7 skills are present (5 existing + 2 new)
   - Confirm categories include 'blockchain' and 'ethical_hacking'

2. **As a regular user:**
   - Register and log in
   - Go to "Create Path"
   - Select "Blockchain & Web3" or "Ethical Hacking & Security"
   - Choose beginner/intermediate/advanced
   - Set hours/days
   - Verify roadmap generates with 10 topics
